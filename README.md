# LLM アプリ開発基礎 — 学習ノート（日本語）

LLM を使ったアプリケーション開発の基礎を、自身の理解のために日本語でまとめた個人的な学習ノートです。

公開されている開発者向け入門コースを受講し、その内容を自身の言葉と構成で整理したメモです。本文は自然な技術書の日本語に訳し、コード・API 名・コマンド・設定値・モデル ID・ボタン名は機能を維持して英語のまま残しています。

| ファイル | テーマ |

| `modules/module-1.md` | 基礎（トークン・コンテキストウィンドウ・モデル階層・プロンプトモード・API 転送） |
| `modules/module-2.md` | プロンプト設計・エージェント・ツール使用・ストリーミング |
