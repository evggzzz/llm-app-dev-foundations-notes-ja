# 演習問題

`modules/` の学習ノートに対するセルフチェック用の演習コンテンツです。ブラウザで開いてそのまま使える HTML 形式で、2 種類あります。

## モジュール別クイズ

公式コースの各モジュール本文から生成した、クエリ駆動（1 概念ブロック → 即 1 問）の自己学習問題集です。アクティブリコールを前提とした設計で、各問に理由欄と💾保存（進捗記憶）付きです。

| ファイル | 対象 | 問数 |
|---|---|---|
| [`module-4-quiz.html`](./module-4-quiz.html) | モジュール 4（プロダクション運用・eval・テスト・セキュリティ） | 188 問 |
| [`module-5-quiz.html`](./module-5-quiz.html) | モジュール 5（再利用化・コントリビューション・デプロイ・信頼境界） | 122 問 |

## 模擬試験の分析

サードパーティの模擬試験を受験した結果の自己分析です。

| ファイル | 内容 |
|---|---|
| [`sundog-exam1/quality-evaluation.html`](./sundog-exam1/quality-evaluation.html) / [.pdf](./sundog-exam1/quality-evaluation.pdf) | 問題集の品質評価（公式 Blueprint との整合性・技術的正確性） |
| [`sundog-exam1/weakness-analysis.html`](./sundog-exam1/weakness-analysis.html) / [.pdf](./sundog-exam1/weakness-analysis.pdf) | 自身の苦手分野の分析と優先復習案 |
| [`sundog-exam2/exam2-review.html`](./sundog-exam2/exam2-review.html) / [.pdf](./sundog-exam2/exam2-review.pdf) | Exam2 の分析 ＋ Exam1/Exam2 統合復習計画（弱点の振り子・誤答18問のトピックグループ化・Prep 優先順位） |

## 使い方

1. HTML ファイルをブラウザ（Safari / Chrome など）で開く
2. クイズ: 問に答える → 正解表示 → 💾保存で進捗を記憶
3. 分析: そのまま閲覧可（リンクで両ファイル相互遷移）

## 注記

- モジュール別クイズは `modules/` と同じ公式コースの内容に基づく個人的な学習用問題です。
- 模擬試験の分析は [Sundog Education（Frank Kane）](https://www.udemy.com/course/claude-certified-developer-foundations-practice-exam-pack/) の有料模擬試験を受験した個人的な振り返りメモです。出題内容の要約・翻訳を含みます。
- 原コンテンツの著作権はそれぞれの原著作者に帰属します。あくまで個人の学習目的の二次メモです。
- 問の正答は学習時点の理解に基づく推定を含みます。正確を期すには各公式ドキュメントをご確認ください。
